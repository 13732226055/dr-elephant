This analysis shows the amount of spill that occurs on mapper side in your code

Example

Mapper Spill

Severity: Critical
Number of spilled records 15000
Number of Mapper output records 7500
Ratio of disk spills to output records 2.0

Suggestions

  1. Use combiner to lower the map output size.
  2. Increase the size of map side sort buffer (io.sort.mb)
  3. Increase the size of index buffer (io.sort.record.percent)
  4. Increase the spill percentage (io.sort.spill.percent), default value is 0.8.
